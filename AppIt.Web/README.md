# AppIt.Web

Frontend project for AppIt (Angular).

## Full Stack Docker Setup

Use the repository root README for complete Docker instructions:
- `../README.md`

## Local Frontend Development

Install dependencies:

```bash
npm install
```

Start dev server with API proxy:

```bash
npm start
```

Run unit tests:

```bash
npm test
```

Run end-to-end tests:

```bash
npm run e2e
```
